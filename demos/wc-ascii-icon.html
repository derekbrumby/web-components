<wc-ascii-icon>

Feed any single-colour SVG path into the component and it will render a text-based mosaic using your chosen character.

The default grid renders 32 columns by 32 rows. Tweak `columns`, `rows`, their singular aliases, or `gap` (alongside `cell-size`) to fine-tune the sampling resolution. The SVG output scales with `--wc-ascii-icon-size`, so the ASCII art stays crisp at any display size.
