# Remote markdown

This document is fetched from a `.md` file next to the demo. It demonstrates:

- Headings & paragraphs
- Lists with **bold** and _italic_ emphasis
- Syntax-highlight ready code blocks:

```js
function greet(name) {
  return `Hello, ${name}!`;
}
```

Tables are supported as well:

| Feature | Status |
| :------ | :----- |
| Sanitization | ✅ |
| Lists | ✅ |
| Tables | ✅ |
